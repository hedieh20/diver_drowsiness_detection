# diver_drowsiness_detection

## Overview
A deep learning-based computer vision system that detects driver drowsiness in real time using facial landmarks and eye state analysis.

## Tools
Python, TensorFlow/Keras, OpenCV, NumPy, Matplotlib

## What I did
- Loaded and preprocessed image dataset
- Built CNN model for classification
- Used data augmentation and class weighting
- Applied early stopping to reduce overfitting
- Evaluated model using confusion matrix and classification report

## Result
Accuracy: ~94%

## Conclusion
This project demonstrates real-time image classification using deep learning.
